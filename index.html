Proyectos

Inteligencia Artificial y visión Artificial

Trabajos realizados en Python con diferentes librerías de procesamiento de imágenes, transformación y tratamiento de datos, IA y demás librerías, dando soluciones para algunas problemáticas y planteando cada uno de estos productos como parte de enseñanza y entendimiento de cada una de las librerías.

Aplicaciones

Trabajos enfocados en desarrollo de aplicaciones móviles Android y de escritorio con lenguajes de Java y .Net, buscando dar productos funcionales para el aprendizaje y la interacción de los diferentes lenguajes.

Frontend

Trabajos en desarrollo de páginas web para muestra de información, implementando diferentes framework que mejoren el desarrollo y optimización de los lenguajes de hipertexto, también diferentes interfases realizadas en con ayuda de herramientas para conseguir una interacción con el usuario mas agradable.

Backend

Trabajos en sistemas de gerenciamiento utilizando frameworks que permitan el uso de patrones como ORM, también optimizando y mejorando el propio relacionamiento entre clases. Sistemas básicos de inserción de datos para bases de datos NoSQL y manipulación de estos por medio del CRUD para bases de datos SQL.
